This crate provides utilities for concurrent programming.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
crossbeam-utils = "0.6"
```

Next, add this to your crate:

```rust
extern crate crossbeam_utils;
```

The minimum required Rust version is 1.26.
